Words cloud on Google Trends

2007 2008 2009

The aim here is to show, with a word cloud, the search trends on Google in the United States, in order to highlight the population fear about 2008 financial crisis.

Features : There will be a word cloud of the (for e.g. 100) most search words in the United States. To interact with the word cloud, we'll use a slider to change the date of the data, so the world cloud will change, but we could also place a play button to go through the dates automatically a smoothly

Here we'll use : Google Trends API to get the data and have a json file (Lecture 4), D3.js to create the word cloud (Lecture 4), and some user interactions (Lecture 5) like a slider to change the date of the data, or the ability to click on a word to see the evolution of the word over time.
