Map of the mortgage default evolution in the United States

The aim here is to show, with a map, the evolution of the mortgage default in the United States during the 2008 financial crisis.

Features : There will be a map of the United States, with the evolution of the mortgage default over time. We'll use a slider to change the date of the data, so the map will change, but we could also place a play button to go through the dates smoothly. We will also be able to change the metric of the map, so we can see the evolution of the mortgage default, the unemployment rate, ...

Here we'll use : a json file for the data (Lecture 4), D3.js to create map (Lecture 4, Lecture 8), and some user interactions (Lecture 5) like a slider to change the date of the data, or the ability to click on a state to see the evolution of the number of mortgage default of this specific state.

Words cloud on Google Trends

The aim here is to show, with a word cloud, the search trends on Google in the United States, in order to highlight the population fear about 2008 financial crisis.

Features : There will be a word cloud of the (for e.g. 100) most searched words in the United States. To interact with the word cloud, we'll use a slider to change the date of the data, so the world cloud will change, but we could also place a play button to go through the dates smoothly.

Here we'll use : Google Trends API to get the data and save it as a json file (Lecture 4), D3.js to create the word cloud (Lecture 4), and some user interactions (Lecture 5) like a slider to change the date of the data, or the ability to click on a word to see the evolution of the word over time.
